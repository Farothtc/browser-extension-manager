## Browser Extension Manager UI

This is a project that aims to reflect a Browser Extension Manager.

### Built With:

- Next.js,
- TypeScript,
- Tailwind CSS,
- daisyUI.

### Used Methods/Hooks:

- useState,
- useEffect,
- filter(),
- map(),
- Conditional Rendering,
- dark: method for instant theme change.
